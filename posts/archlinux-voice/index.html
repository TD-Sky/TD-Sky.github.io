折腾ArchLinux的声卡

写于 2022-01-19
  • 声卡驱动
  • 音频管理器

声卡驱动

$ sudo pacman -S sof-firmware

音频管理器

# 本体 及 会话/策略管理器
$ sudo pacman -S pipewire wireplumber

# 替代老牌音频调节器的前端
$ sudo pacman -S pipewire-pulse pipewire-alsa pipewire-jack

重开终端，启用相关服务：

# 启用Pipewire相关服务：
$ systemctl enable pipewire --user
$ systemctl enable pipewire-pulse --user
